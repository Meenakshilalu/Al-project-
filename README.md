# AI-Powered Medical Advisor

An intelligent healthcare system that provides AI-based medical guidance,
doctor consultations, and pharmacy management.

## Features
- User symptom analysis
- Doctor appointment booking
- Pharmacist medicine management
- Secure login system
- Admin monitoring dashboard

## Technologies Used
- Python
- Django
- SQLite3
- HTML, CSS, JavaScript

## Modules
- Admin
- User
- Doctor
- Pharmacist

## How to Run
1. Clone the repository
2. Install requirements
3. Run migrations
4. Start the server
